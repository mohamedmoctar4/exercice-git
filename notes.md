# Ce que j'ai retenue

## Git init
Cree un nouveau depot git dans le dossier courant
## Git status
Affiche l'etat des fichiers(modifies, prepares, non suivis)
## Git add fichier
Prepare un fichier pour le prochain commit
## Git commit -m msg
Sauvegarde les changements prepares avec un message(doit se faire apres  git add)
## Git log
Affiche l'historique complet des commits
## Git log --oneline
Affiche l'historique en version compacte
## Git push
Envoie les commits vers github
## Git config --global ...
Configure git (nom, email)
